# ДИП : КОД Gazprom GoRest Control

Тестовое приложение на **React + TypeScript** для работы с API [GoRest](https://gorest.co.in/).

## Выполненные требования

- обязательное поле `access token` с валидацией на непустое значение;
- два режима главной страницы: список пользователей и список постов;
- таблица пользователей с именем, фамилией, email и статусом;
- таблица постов с ID, заголовком и автором;
- пагинация: номера страниц, кнопки «Предыдущая» / «Следующая», выбор 10 / 25 / 50 элементов;
- переход по строке таблицы в карточку пользователя или поста;
- карточка поста показывает все комментарии;
- карточка пользователя показывает данные пользователя и его посты;
- возврат из карточки обратно к списку.

## Дополнительные фичи

- сохранение рабочего состояния в `localStorage`;
- локальный фильтр по текущей странице таблицы;
- избранные пользователи и посты;
- история последних открытых карточек;
- сохранение access token между перезагрузками страницы;
- раздельное сохранение страницы и размера страницы для пользователей и постов;
- кнопка сброса сессии, которая очищает локально сохраненные настройки;
- hash-навигация для прямого открытия карточек пользователя или поста.

## localStorage

В браузере сохраняются:

- access token;
- выбранный режим (`users` или `posts`);
- страницы и размер страницы для каждого режима;
- строка локального фильтра;
- избранные пользователи и посты;
- история последних открытых карточек.

Кнопка «Сбросить локальные данные» очищает состояние приложения и возвращает его к стартовому виду.

## Архитектура

Проект собран по простой слоистой схеме.

- `pages/` хранит экраны приложения;
- `components/` хранит UI-компоненты;
- `hooks/` хранит переиспользуемую логику и работу с состоянием;
- `api.ts` изолирует работу с GoRest;
- `constants/`, `lib/` и `types.ts` отделяют конфигурацию, чистые функции и типы от JSX.
- `localStorage` вынесено в отдельный хук `usePersistedWorkspace`.

```text
src/
  api.ts                         # клиент GoRest API
  App.tsx                        # корневой компонент
  pages/
    DashboardPage.tsx            # главный экран приложения
  components/
    EmptyState.tsx               # общий UI-блок
    dashboard/                   # UI-блоки главного экрана
  hooks/
    useListData.ts               # загрузка таблиц
    useDetailData.ts             # загрузка карточек
    usePersistedWorkspace.ts     # localStorage и настройки рабочего состояния
    useHashRoute.ts              # простая hash-навигация
    useLocalStorage.ts           # базовый hook localStorage
  constants/
    dashboard.ts                 # page size, storage keys, defaults
  lib/
    dashboardUtils.ts            # чистые helper-функции
  styles/
    consta-theme.css             # импорты темы Consta
  types.ts                       # типы API, маршрутов и состояния экрана
```

## Как пользоваться

1. Получите access token в GoRest.
2. Вставьте токен в поле `Access token`.
3. Переключайтесь между пользователями и постами.
4. Меняйте размер страницы, переходите по страницам и открывайте строки таблицы.
5. Добавляйте важные записи в избранное - они сохранятся после перезагрузки.
